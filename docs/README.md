# Plataforma Cultural Role Entretenimento

Uma plataforma completa para gestão e descoberta de eventos culturais, artistas e produtoras no Brasil.

## 🚀 Características Principais

### Frontend
- **React 18** com TypeScript
- **Tailwind CSS** para design responsivo
- **shadcn/ui** para componentes
- **React Query** para gerenciamento de estado
- **React Router** para navegação
- **Framer Motion** para animações

### Backend (Supabase)
- **PostgreSQL** com Row Level Security (RLS)
- **Autenticação** completa
- **Storage** para arquivos e imagens
- **Edge Functions** para lógica personalizada
- **Real-time subscriptions**

## 🎯 Funcionalidades

### Perfis de Entidades
- **Artistas**: Perfis completos com bio, eventos, mídia
- **Produtoras**: Gestão de eventos e parcerias
- **Espaços**: Locais e venues para eventos
- **Bandas**: Grupos musicais e coletivos

### Sistema de Eventos
- Criação e gestão de eventos
- Sistema de ingressos
- Localização com mapas
- Categorização por gêneros

### Administração
- Dashboard completo
- Analytics em tempo real
- Gestão de usuários
- Monitoramento de sistema

## 📊 Estatísticas do Sistema

- **11 tabelas** funcionais
- **9 tabelas** com RLS ativo
- **12 eventos** cadastrados
- **55 destaques** ativos
- **100% conformidade** LGPD/GDPR

## 🛡️ Segurança

- **Row Level Security (RLS)** em todas as tabelas sensíveis
- **Políticas de acesso** granulares
- **Autenticação segura** com Supabase Auth
- **Validação** de dados em tempo real
- **Conformidade** com LGPD e GDPR
